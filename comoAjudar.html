Como ajudar?

O jogo é completamente gratúito, porém infelizente existe uma taxa para manter um site na web. Por conta disso resolvemos abrir para os usuários divulgarem empresas, negócios, eventos, etc. Estamos a procura de apoiadores, pensando nisso criamos dois tipos de planos com valores simbólicos, pensados apenas para manter o site de pé!

  • Anúncio simples: R$10,00/mês ou R$50,00 até o final do ano (31 de dezembro)

    • O anúncio simples inclúi:

    1. Uma imagem ocupando 10% de cada tela com slogam e link direcionando o usuário para o site oficial ou rede social. Segue um exemplo.
    2. Um espaço ilimitado para divulgação personalizada e descrição completa na área dos apoiadores. Segue um exemplo
  • Anúncio premium: R$20,00/mês ou R$100,00 até o final do ano (31 de dezembro)

    • O anúncio premium inclúi:

    1. Uma imagem ocupando 20% de cada tela com slogam e link direcionando o usuário para o site oficial ou rede social. Segue um exemplo.
    2. Um espaço ilimitado para divulgação personalizada e descrição completa na área dos apoiadores. Segue um exemplo
    3. Uma divulgação exclusiva a sua escolha todo mês no nosso instagram.
